Installation

Run the following commands and open the project in Android Studio.

$ https://github.com/tommy351/ehreader-android.git
$ cd ehreader-android
$ git submodule update --init
$ cd android-stackblur/StackBlur
$ ndk-build

Generating DAO

This app uses GreenDAO as ORM. Before you get started or after the DAO is updated, you have to run the gradle task DaoGenerator:run.
